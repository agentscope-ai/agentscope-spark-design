## Installation

### Install using npm

> Installation methods for yarn, pnpm, and bun are not repeated here. If your network environment is poor, we recommend using <a href="https://github.com/cnpm/cnpm" target="_blank" rel="noopener noreferrer">cnpm</a>.

```bash
$ npm install antd --save
$ npm install @agentscope-ai/design --save
$ npm install @agentscope-ai/chat --save
```

> Note: If you are an Alibaba internal developer, please use `tnpm install @ali/agentscope-ai-chat @ali/agentscope-ai-design --save`

## Tree Shaking Support

`@agentscope-ai/chat` supports tree shaking based on ES modules by default.

## TypeScript

`@agentscope-ai/chat` is written in TypeScript and provides complete definition files.

## Open Source

`@agentscope-ai/design` and `@agentscope-ai/chat` are actively progressing towards open source, with completion expected by the end of 2025. Currently, you can access our projects through npm.
