

# Vibe Coding

如何让 Cursor、Claude Code 等工具理解 Spark Chat。我们支持 LLMs.txt 文件，使 Spark Chat 的文档可供大型语言模型使用。

## 目录概览

以下文件可供使用：


- [index.llms.txt](https://g.alicdn.com/code/npm/@ali/agentscope-ai-chat/1.1.13/llms/index.llms.txt)：主要的 LLMs.txt 目录文件
- [all.llms.txt](https://g.alicdn.com/code/npm/@ali/agentscope-ai-chat/1.1.13/llms/all.llms.txt)：Spark Chat 的完整文档

## 使用方法

### Cursor

在 Cursor 中使用 @Docs 功能来将 LLMs.txt 文件包含到你的项目中。[阅读更多](https://docs.cursor.com/en/context/@-symbols/@-docs)
